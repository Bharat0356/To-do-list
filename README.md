# JS TODO LIST

A simple to-do list application built using HTML, CSS, JavaScript, and Bootstrap.

## Features

- Add new tasks to the to-do list
- Mark tasks as completed
- Remove tasks from the list

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap
